# librarian
An alternative frontend for LBRY/Odysee. Inspired by Invidious.

## Features
* Copylefted libre software under the AGPL
* Lightweight
* No ads
* No tracking
* No crypto garbage
* JavaScript not required
* No Code of Conduct
* No Contributor License Agreement or Developer Certificate of Origin
